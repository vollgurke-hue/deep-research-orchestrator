## How to Use Templates

1. **Copy the relevant template** from this file
2. **Fill in the placeholders** (remove brackets)
3. **Save in appropriate location:**
   - Base Research: `2-working-state/0-base-research/[category]/[sub-category].md`
   - Iterations: `2-working-state/2-iterations/`
   - Output: `3-output/`

4. **Update regularly** (add date to "Last Updated")
