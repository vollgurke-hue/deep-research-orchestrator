## Quick Start

```bash
# 1. Clone and setup
git clone <repo>
cd deep-research-orchestrator

# 2. Install dependencies
pip install -e .

# 3. Setup llama.cpp
./scripts/setup_llama_cpp.sh

# 4. Download models
./scripts/download_models.sh

# 5. Start GUI
python gui/app.py
```
